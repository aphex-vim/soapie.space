What is Walkies?

Walkies is a series put on by popular youtuber, Simpleflips, where he is attempting to achieve S rank in Super Mario Maker 2 versus without ever using the run button.

Why am I doing this?

I'm studying programming in school and I find using it to analyze data to be pretty fun. Previously, this data was being kept by another community member in this spreadsheet, but they stopped at episode 21. I want to use my programming experience to automate as much of the process of keeping this information as possible, and to satisfy all the other Simpleflips viewers that found the previous spreadsheet interesting.

Interested in doing your own analysis?

  • All of the scripts I use to record and display this information are available in this website's GitHub repository found here.
  • If you want just to download the latest csv file used to generate these tables and graphs, you can download it here.
